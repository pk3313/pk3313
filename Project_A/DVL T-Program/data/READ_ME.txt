CF1xxxx : Automatic Ensemble Cycling : 현재 주기가 완료된 후 다음 데이터 수집 주기를 자동으로 시작합니다.
CF0xxxx : Manual Ensemble Cycling : 데이터 앙상블 전송 후 STANDBY 모드로 들어가 "<" 프롬프트를 표시하고 새 명령을 기다립니다. 
CFx1xxx : Automatic Ping Cycling : 준비되면 즉시 Ping을 실행합니다.
CFx0xxx : Manual Ping Cycling : ">" 문자를 표시해 Ping 준비 신호를 보낸 다음, ping을 수행하기 전에 <ENTER>를 수신할 때까지 기다립니다. 
CFxx2xx : Hex-ASCII Data output, Carriage Return-Linefeed  delimited : serial output이 활성화 된 경우, 각 앙상블 끝에 캐리지 리턴 라인 피드가 있는 판독 가능한 16진수 ASCII 형식으로 전송합니다.
CFxx1xx : Binary Data Output : serial output이 활성화 된 경우, 이진 형식(Binary)으로 전송합니다. 
CFxx0xx : Hex-ASCII Data Output : serial output이 활성화 된 경우, 16진수 ASCII 형식으로 전송합니다.
CFxxx1x : Enable Serial Output : RS-232/422 Serial interface로 전송합니다. 
CFxxx0x : Disable Serial Output : RS-232/422 interface로 전송되지 않습니다.
CFxxxx1 : Enable Data Recorder : 데이터를 recorder에 저장합니다.
CFxxxx0 : Disable Data Recorder : 데이터를 recorder에 저장하지 않습니다.
